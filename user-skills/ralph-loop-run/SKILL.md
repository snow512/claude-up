# 랄프루프 실행

사용자가 "랄프루프해"와 함께 작업 내용을 입력하면, ralph-loop 플러그인의 `/ralph-loop` 커맨드를 사용하여 해당 작업을 **10회 반복** 실행한다.

## 실행 방법

1. 사용자 입력에서 **실행할 작업 내용**을 추출한다
2. `/ralph-loop` 슬래시 커맨드를 호출한다:

```
/ralph-loop "<작업 내용>" --max-iterations 10
```

## 예시

사용자: "랄프루프해 전체 코드 리뷰하고 개선해"
→ `/ralph-loop "전체 코드 리뷰하고 개선해" --max-iterations 10`

사용자: "랄프루프해 테스트 커버리지 올려"
→ `/ralph-loop "테스트 커버리지 올려" --max-iterations 10`

## 주의사항

- 반복 횟수는 기본 10회. 사용자가 횟수를 지정하면 해당 횟수로 변경
  - 예: "랄프루프 5번 돌려 코드정리해" → `--max-iterations 5`
- 작업 내용이 명확하지 않으면 사용자에게 확인 후 실행
- completion-promise가 필요하면 작업 내용에서 자동 추출하거나 적절히 설정
